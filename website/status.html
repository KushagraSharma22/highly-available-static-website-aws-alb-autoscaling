Note: This dashboard queries http://169.254.169.254/latest/meta-data/ using IMDSv2 directly from the browser. It only resolves with real values when this page is served from an Apache instance running on Amazon EC2, inside that instance's own network. On any other host — including local previews — every field falls back to "Available after deployment on AWS EC2."
